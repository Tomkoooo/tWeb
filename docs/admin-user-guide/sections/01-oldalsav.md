### Plugin menüpontok

Ha a telepítésedben engedélyezett pluginok vannak, az oldalsávban megjelenik a **Pluginok** szekció. Minden plugin saját almenüvel rendelkezik (pl. tábor foglalás, sajtóanyagok). A plugin fejezetekben részletesen leírjuk ezeket.

### Ha a webshop ki van kapcsolva

Tábor- vagy landing-only telepítéseknél (pl. Minecraft tábor) **nem jelennek meg** a termék- és rendelés menüpontok. Ehelyett:

- Az **Áttekintés** a plugin főoldalára irányíthat
- A **CMS** és **Kapcsolat** továbbra is elérhető
- A **Súgó** mindig elérhető marad
