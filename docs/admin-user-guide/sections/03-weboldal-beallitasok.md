## Hol találod?

[/admin/cms/settings](/admin/cms/settings) — vagy a [/admin/cms](/admin/cms) hub **Weboldal beállítások** kártyái.

Ezek a beállítások **az egész weboldalra** vonatkoznak, függetlenül attól, melyik CMS oldalt szerkeszted éppen.

## Téma

**Link:** [/admin/cms/settings?section=theme](/admin/cms/settings?section=theme)

| Mit állíthatsz? | Hatás |
| --- | --- |
| Színek (elsődleges, háttér, kiemelés) | Az egész honlap színvilága |
| Tipográfia | Betűtípusok, méretek |
| Kerekítések, árnyékok | Gombok, kártyák megjelenése |

**Mikor használd:** márkafrissítés, szezonális színváltás, olvashatóság javítása.

## SEO

**Link:** [/admin/cms/settings?section=seo](/admin/cms/settings?section=seo)

| Mező | Mit jelent? |
| --- | --- |
| Alapértelmezett oldalcím | Google keresésben megjelenő cím, ha az oldalnak nincs sajátja |
| Meta leírás | Rövid összefoglaló a keresőben |
| Favicon | Böngésző fül ikonja |
| Megosztási kép (OG) | Facebook / LinkedIn link előnézet képe |

**Mikor használd:** új domain, rebranding, jobb keresőoptimalizálás.

## Márka és logók

**Link:** [/admin/cms/settings?section=branding](/admin/cms/settings?section=branding)

| Mező | Mit jelent? |
| --- | --- |
| Weboldal / bolt neve | Fejlécben, e-mailekben megjelenő név |
| Navbar logó | Felső sáv logója |
| Lábléc logó | Lábléc logója (ha külön van) |

## Lábléc

**Link:** [/admin/cms/settings?section=footer](/admin/cms/settings?section=footer)

| Mit állíthatsz? | Hatás |
| --- | --- |
| Lábléc szövegek | Copyright, rövid leírás |
| Gyors linkek | Lábléc menüpontok |
| Közösségi média | Facebook, Instagram stb. linkek |

## Kapcsolat e-mailek

**Link:** [/admin/cms/settings?section=contact](/admin/cms/settings?section=contact)

| Mező | Mit jelent? |
| --- | --- |
| Megjelenő e-mail cím | Amit a látogatók látnak a honlapon |
| Kapcsolat űrlap címzettjei | Hova érkeznek a [/admin/contact](/admin/contact) üzenetek |
| Számlázási hiba értesítések | Belső értesítés, ha a számlázás hibázik |

> A főoldal **Kapcsolat** blokkjában lévő űrlap üzenetei ide beállított címzettekhez érkeznek.

## Munkafolyamat

1. Nyisd meg a kívánt szekciót a fenti linkeken
2. Módosítsd a mezőket
3. **Mentés** → **Közzététel** (ugyanúgy, mint a CMS oldalaknál)
4. Ellenőrizd a nyilvános honlapon
