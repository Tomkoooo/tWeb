## Popup bannerek

**Belépés:** [/admin/cms/popups](/admin/cms/popups)

Popupok rövid marketing üzenetek, amelyek **felugró ablakként** jelennek meg bizonyos oldalakon.

### Mit állíthatsz be?

| Mező | Mit jelent? |
| --- | --- |
| Cím / szöveg | A popup tartalma |
| Kép | Háttér- vagy illusztrációs kép |
| Gomb szöveg és link | Hova visz a CTA (pl. akció, termék) |
| Cél URL-ek | Mely oldalakon jelenjen meg (főoldal, bolt, kategória, termék) |
| Aktív / inaktív | Ki-be kapcsolás kampány időszakokra |

### Munkafolyamat

1. [/admin/cms/popups](/admin/cms/popups) → **Új popup**
2. Töltsd ki a tartalmat és válaszd ki a megjelenési oldalakat
3. Mentsd és aktiváld
4. Ellenőrizd a nyilvános oldalon

## Kapcsolat üzenetek

**Belépés:** [/admin/contact](/admin/contact)

Itt érkeznek be a honlap **kapcsolatfelvételi űrlapján** küldött üzenetek.

### Mit látsz?

- Feladó neve és e-mail címe
- Üzenet szövege
- Beküldés ideje

### Fontos

A címzetteket a **Kapcsolat e-mailek** beállításoknál adod meg: [/admin/cms/settings?section=contact](/admin/cms/settings?section=contact)

## Jogi dokumentumok (Beállítások)

**Belépés:** [/admin/info](/admin/info) → **Jogi dokumentumok** szekció

Itt tölthetsz fel PDF fájlokat:

| Dokumentum | Tipikus tartalom |
| --- | --- |
| **Impresszum** | Cégadatok, felelős kiadó |
| **ÁSZF** | Általános szerződési feltételek |
| **GDPR / Adatvédelem** | Adatkezelési tájékoztató |

**Lépések:**

1. Készítsd el a PDF-et
2. [/admin/info](/admin/info) → válaszd ki a dokumentum típusát
3. Töltsd fel a fájlt
4. A honlapon a lábléc linkjei a feltöltött PDF-re mutatnak
