# Kerámia Dental

Landing template for **Kerámia Dental** summer campaigns in Székesfehérvár.

## Design intent

- Cream, gold and charcoal palette (Playfair Display + Space Grotesk) aligned with the standalone campaign demos.
- **Home** (`/`): campaign hub linking to all three offers (falls back when homepage CMS blocks are empty).
- **Campaign landings** (static pages):
  - `/fogfeherites` — teeth whitening + scaling (Hungarian)
  - `/implant` — implants & prosthetics (Hungarian)
  - `/newpatient` — New Patient Special (English)
- Shared `CampaignLanding` layout: hero, benefits, offer, process, services, before/after slider, FAQ, contact form.

Deployment: `landing` (`DEPLOYMENT_KEY=keramia-dental`).
