# Erdweg template

Landing-only template for **Erdweg** — építőipari családi vállalkozás. Industrial charcoal/ember visual language adapted from the Ferro & Sons mockup: hero, services, stats, projects, testimonials, and contact inquiry form (no webshop).

Deployment key: `erdweg` (`ENABLE_SHOP=false` recommended).

Activate at `/admin/templates` after setting `DEPLOYMENT_KEY=erdweg`. Edit homepage copy and images at `/admin/cms/home`. Configure contact form recipients under **Weboldal beállítások → Kapcsolat e-mailek**.
