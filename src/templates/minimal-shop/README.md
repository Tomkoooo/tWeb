# minimal-shop

A second template demonstrating the contract: light editorial aesthetic with serif typography, generous whitespace, and a curated-feel product grid. Suitable for boutiques and considered catalogs.

- Theme tokens swap the dark palette for a warm light scheme; admin can override individually.
- Chrome (`Navbar`, `Footer`) is hand-built — no shared chrome with default-modern.
- Home page schema is purpose-built (hero / pillars / featured / closing) rather than a generic block list, illustrating that templates own their content shape.
- Shop and PDP pages render the same product data through a different layout.

This template is the proof that the engine isn't just "themable colors". A template can replace the entire UI of every public page while reusing all the underlying data and admin tooling.
